ol(Ordered List) : 순서가 있는 목록

               -type 속성 : 표시되는 숫자 형식 지정

                type="a" :영어 소문자(a,b,c,,,,)
                type="A" : 영어 대문자(A,B,C,,,)
                type="i" : 로마 소문자(i,ii,,,v,,,x)
                type="I" : 로마 대문자(I,II,,,V,,,X)
                start 속성 : 목록 시작 번호

                reversed : 순서 뒤집기(역순)
